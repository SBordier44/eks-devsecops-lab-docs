# ADR-0001 Repository Structure

## Context
The project is structured using multiple repositories to separate concerns.

## Decision
Four repositories are used:

- eks-devsecops-lab-infra
- eks-devsecops-lab-app
- eks-devsecops-lab-gitops
- eks-devsecops-lab-docs

## Consequences
This structure mirrors real-world DevSecOps platforms where:

- infrastructure
- application code
- deployment configuration
- documentation

are managed independently.
